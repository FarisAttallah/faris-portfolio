## ğŸ“¦ Getting Started

1. **Clone the repo:**
   ```sh
   git clone https://github.com/FarisAttallah/faris-portfolio.git
   cd faris-portfolio
   ```
2. **Install dependencies:**
   ```sh
   npm install
   # or
   yarn install
   ```
3. **Run the development server:**
   ```sh
   npm run dev
   # or
   yarn dev
   ```
   Open [http://localhost:3000](http://localhost:3000) in your browser.
